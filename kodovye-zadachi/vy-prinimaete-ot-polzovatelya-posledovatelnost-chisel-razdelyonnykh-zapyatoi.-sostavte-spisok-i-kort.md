# Вы принимаете от пользователя последовательность чисел, разделённых запятой. Составьте список и корт

```python
# Получение ввода от пользователя
user_input = input("Введите последовательность чисел, разделённых запятой: ")

# Разделение введённой строки на список чисел
numbers_list = [int(num) for num in user_input.split(',')]

# Создание кортежа из списка
numbers_tuple = tuple(numbers_list)

# Вывод результата
print("Список чисел:", numbers_list)
print("Кортеж чисел:", numbers_tuple)
```

Этот код просит пользователя ввести последовательность чисел, разделённых запятой. Затем он разбивает введённую строку на список чисел и создаёт кортеж из этого списка. Выводятся исходный список и кортеж. Обратите внимание, что предполагается, что пользователь вводит целые числа, и код не включает проверку на ошибки ввода.
