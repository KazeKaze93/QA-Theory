# Экранирование символов

Экранирование символов - это процесс представления специальных символов в строке с помощью обратной косой черты (`\`), чтобы указать, что следующий символ имеет специальное значение или требует особой интерпретации. В Java экранирование символов часто используется в строках для представления специальных символов или последовательностей, таких как символы перевода строки (), табуляции (), одинарные кавычки (`\'`), двойные кавычки (`\"`), обратные косые черты (`\\`) и другие.

* (\n) : новая строка&#x20;
* (\t) : табуляция&#x20;
* `\'`: одиночная кавычка
* `\"`: двойная кавычка
* `\\`: обратная косая черта
* (\r) : возврат каретки (переход к началу строки)
* `\b`: удаление (забой, backspace)
* `\f`: разрыв страницы (перевод формата)

Примеры использования экранирования символов в Java:

```java
String newline = "This is a new line.\nThis is the second line."; // Новая строка
String tabbed = "First\tSecond\tThird"; // Табуляция
String singleQuote = "It's raining"; // Одиночная кавычка
String doubleQuote = "\"Hello, World!\""; // Двойная кавычка
String backslash = "C:\\Users\\Username\\Documents"; // Обратные косые черты для пути к файлу
```
