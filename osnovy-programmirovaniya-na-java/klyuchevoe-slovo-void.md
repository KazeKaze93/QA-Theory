# Ключевое слово void

Ключевое слово `void` в Java используется в сигнатуре метода для обозначения того, что метод не возвращает какое-либо значение после выполнения. Это означает, что метод выполняет определенные операции, но не возвращает никаких данных.

Пример сигнатуры метода с возвращаемым значением `void`:

```java
public void printMessage(String message)
```

Этот метод `printMessage` принимает аргумент типа `String` и не возвращает никаких данных после выполнения. Вместо этого он может выполнить операцию вывода сообщения на экран или выполнить другие действия, но ничего не возвращает обратно вызывающему коду.

Использование ключевого слова `void` в сигнатуре метода является обязательным, если метод не должен возвращать какое-либо значение. Если метод должен вернуть какое-то значение, в сигнатуре метода указывается соответствующий тип данных возвращаемого значения, например `int`, `double`, `String` и т. д.
