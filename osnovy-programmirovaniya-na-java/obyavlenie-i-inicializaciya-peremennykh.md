# Объявление и инициализация переменных

В Java переменные создаются с использованием синтаксиса объявления переменных. Объявление переменной включает в себя указание типа данных переменной, её имени и, при необходимости, начального значения. Вот основные способы создания переменных в Java:

1. **Объявление и инициализация переменных**: Вы можете объявить переменную и инициализировать её значением в одной строке.

```java
int x = 10; // Объявление переменной типа int с именем x и начальным значением 10
String name = "John"; // Объявление переменной типа String с именем name и начальным значением "John"
```

2. **Объявление переменных без инициализации**: Вы также можете объявить переменную без назначения ей начального значения. В этом случае, переменная будет иметь значение по умолчанию в зависимости от типа данных (например, 0 для числовых типов, false для boolean и null для ссылочных типов).

```java
int y; // Объявление переменной типа int с именем y (без инициализации)
```

3. **Изменение значения переменной**: Значение переменной можно изменить путём присваивания нового значения.

```java
x = 20; // Изменение значения переменной x на 20
```

4. **Инициализация переменных позже**: Вы также можете объявить переменную и присвоить ей значение позже в коде.

```java
int z; // Объявление переменной типа int с именем z
z = 30; // Присваивание значения переменной z позже
```

5. **Одновременное объявление нескольких переменных**: Вы можете одновременно объявить несколько переменных одного типа.

```java
int a, b, c; // Объявление трех переменных типа int с именами a, b и c
```
