# Работа с ArrayList

&#x20;

1.  **Создание `ArrayList`**: Вы можете создать новый объект `ArrayList` с помощью конструктора без аргументов или с указанием начальной емкости:

    ```java
    ArrayList<String> list = new ArrayList<>();
    ArrayList<Integer> numbers = new ArrayList<>(10); // с начальной емкостью 10
    ```
2.  **Добавление элементов**: Добавление элемента в конец списка осуществляется методом `add()`:

    ```java
    list.add("Элемент 1");
    list.add("Элемент 2");
    ```
3.  **Получение элементов**: Получение элемента из списка по индексу выполняется методом `get()`:

    ```java
    String element = list.get(0); // Получить первый элемент
    ```
4.  **Удаление элементов**: Удаление элемента из списка по индексу или по самому элементу:

    ```java
    list.remove(0); // Удалить первый элемент
    list.remove("Элемент 2"); // Удалить элемент "Элемент 2"
    ```
5.  **Изменение элементов**: Изменение элемента списка по индексу:

    ```java
    list.set(0, "Новый элемент"); // Заменить первый элемент на "Новый элемент"
    ```
6.  **Размер списка**: Получение размера списка с помощью метода `size()`:

    ```java
    int size = list.size();
    ```
7.  **Проверка наличия элемента**: Проверка наличия элемента в списке с помощью метода `contains()`:

    ```java
    boolean containsElement = list.contains("Элемент 1");
    ```
8.  **Итерация по списку**: Вы можете перебирать элементы списка с помощью цикла for-each или цикла по индексу:

    ```java
    for (String element : list) {
        System.out.println(element);
    }

    for (int i = 0; i < list.size(); i++) {
        String element = list.get(i);
        System.out.println(element);
    }
    ```
