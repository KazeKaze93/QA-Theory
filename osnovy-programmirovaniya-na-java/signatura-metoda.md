# Сигнатура метода

Сигнатура метода - это его уникальная идентификационная схема, которая включает в себя имя метода, типы его параметров (их порядок и количество) и тип возвращаемого значения.

Вот пример общего формата сигнатуры метода в Java:

```java
[модификатор_доступа] [тип_возвращаемого_значения] имя_метода([список_параметров])
```

Примеры сигнатур методов в Java:

1. Метод без параметров и возвращаемого значения:

```java
public void printHello()
```

2. Метод с параметрами и возвращаемым значением:

```java
public int sum(int a, int b)
```

3. Метод с параметрами, но без возвращаемого значения:

```java
private void showMessage(String message)
```

4. Метод с параметрами разных типов и возвращаемым значением:

```java
protected double calculateAverage(int[] numbers)
```
