# Условные операторы

Условные операторы в Java используются для принятия решений на основе различных условий. Они позволяют выполнить определенный блок кода, если условие истинно, и выполнить другой блок кода, если условие ложно.&#x20;

1. **Оператор if**: Оператор `if` используется для выполнения блока кода, если заданное условие истинно.

```java
if (условие) {
    // Код, который будет выполнен, если условие истинно
}
```

Пример:

```java
int x = 10;
if (x > 5) {
    System.out.println("x больше 5");
}
```

2. **Оператор if-else**: Оператор `if-else` используется для выполнения одного блока кода, если условие истинно, и другого блока кода, если условие ложно.

```java
if (условие) {
    // Код, который будет выполнен, если условие истинно
} else {
    // Код, который будет выполнен, если условие ложно
}
```

Пример:

```java
int x = 3;
if (x > 5) {
    System.out.println("x больше 5");
} else {
    System.out.println("x меньше или равен 5");
}
```

3. **Оператор if-else if-else**: Оператор `if-else if-else` используется для выполнения одного из нескольких блоков кода, в зависимости от условия.

```java
if (условие1) {
    // Код, который будет выполнен, если условие1 истинно
} else if (условие2) {
    // Код, который будет выполнен, если условие2 истинно
} else {
    // Код, который будет выполнен, если все предыдущие условия ложны
}
```

Пример:

```java
int x = 3;
if (x > 5) {
    System.out.println("x больше 5");
} else if (x < 5) {
    System.out.println("x меньше 5");
} else {
    System.out.println("x равен 5");
}
```
