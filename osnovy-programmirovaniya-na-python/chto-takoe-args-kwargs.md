# –ß—Ç–æ —Ç–∞–∫–æ–µ \*args, \*\*kwargs?

–í Python `*args` –∏ `**kwargs` ‚Äî —ç—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏, –ø–æ–∑–≤–æ–ª—è—é—â–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –ø—Ä–∏–Ω–∏–º–∞—Ç—å **–ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤**.

***

### **1. `*args` ‚Äî –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤**

* –°–æ–±–∏—Ä–∞–µ—Ç –≤—Å–µ **–ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã** –≤ **–∫–æ—Ä—Ç–µ–∂**.
* –ò–º—è `args` —É—Å–ª–æ–≤–Ω–æ–µ ‚Äî –º–æ–∂–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å `*values`, `*items` –∏ —Ç.–ø., –Ω–æ –∑–≤—ë–∑–¥–æ—á–∫–∞ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–∞.

```python
def my_func(*args):
    print(args)

my_func(1, 2, 3)  
# (1, 2, 3)
```

–ú–æ–∂–Ω–æ –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞—Ç—å—Å—è:

```python
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # 10
```

***

### **2. `**kwargs` ‚Äî –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤**

* –°–æ–±–∏—Ä–∞–µ—Ç –≤—Å–µ **–∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã** –≤ **—Å–ª–æ–≤–∞—Ä—å**.
* –ò–º—è `kwargs` ‚Äî —É—Å–ª–æ–≤–Ω–æ–µ, –º–æ–∂–Ω–æ –Ω–∞–∑–≤–∞—Ç—å –∫–∞–∫ —É–≥–æ–¥–Ω–æ, –Ω–æ —Å `**`.

```python
def my_func(**kwargs):
    print(kwargs)

my_func(name="Kakha", age=30)  
# {'name': 'Kakha', 'age': 30}
```

–ú–æ–∂–Ω–æ –æ–±—Ä–∞—â–∞—Ç—å—Å—è –ø–æ –∫–ª—é—á–∞–º:

```python
def greet(**kwargs):
    print(f"Hello, {kwargs['name']}!")

greet(name="Kakha")  # Hello, Kakha!
```

### **3. –ö–æ–º–±–∏–Ω–∞—Ü–∏—è `*args` –∏ `**kwargs`**

* –ú–æ–∂–Ω–æ –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∏ –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–µ, –∏ –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –≤–º–µ—Å—Ç–µ.
* –ü–æ—Ä—è–¥–æ–∫ –≤–∞–∂–µ–Ω: –æ–±—ã—á–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã ‚Üí `*args` ‚Üí –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Å –¥–µ—Ñ–æ–ª—Ç–æ–º ‚Üí `**kwargs`.

```python
def func(a, b, *args, c=10, **kwargs):
    print(f"a={a}, b={b}, args={args}, c={c}, kwargs={kwargs}")

func(1, 2, 3, 4, 5, c=20, x=100, y=200)
# a=1, b=2, args=(3, 4, 5), c=20, kwargs={'x': 100, 'y': 200}
```

***

### **4. –†–∞—Å–ø–∞–∫–æ–≤–∫–∞ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏**

* `*` —Ä–∞—Å–ø–∞–∫–æ–≤—ã–≤–∞–µ—Ç **—Å–ø–∏—Å–∫–∏/–∫–æ—Ä—Ç–µ–∂–∏** –≤ –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã.
* `**` —Ä–∞—Å–ø–∞–∫–æ–≤—ã–≤–∞–µ—Ç **—Å–ª–æ–≤–∞—Ä–∏** –≤ –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã.

```python
def show(a, b, c):
    print(a, b, c)

nums = [1, 2, 3]
show(*nums)  # 1 2 3

params = {"a": 1, "b": 2, "c": 3}
show(**params)  # 1 2 3
```

* –ü–æ—á–µ–º—É `*args` –∏–¥—ë—Ç –ø–µ—Ä–µ–¥ `**kwargs` –≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ —Ñ—É–Ω–∫—Ü–∏–∏?\
