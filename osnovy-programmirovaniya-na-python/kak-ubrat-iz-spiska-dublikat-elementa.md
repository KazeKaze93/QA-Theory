### **1. –ß–µ—Ä–µ–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–æ (`set`)**

* –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ–º —Å–ø–∏—Å–æ–∫ –≤ `set`, –∞ –ø–æ—Ç–æ–º –æ–±—Ä–∞—Ç–Ω–æ –≤ `list`.
* –ú–∏–Ω—É—Å: **—Ç–µ—Ä—è–µ—Ç—Å—è –ø–æ—Ä—è–¥–æ–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤**.


### **2. –ß–µ—Ä–µ–∑ `dict.fromkeys()`**

* –°–æ—Ö—Ä–∞–Ω—è–µ—Ç **–ø–æ—Ä—è–¥–æ–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤** (Python 3.7+).

```python

```

### **3. –ß–µ—Ä–µ–∑ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä —Å –ø—Ä–æ–≤–µ—Ä–∫–æ–π**

* –ú–æ–∂–Ω–æ –≤—Ä—É—á–Ω—É—é –ø—Ä–æ–≤–µ—Ä—è—Ç—å, —á—Ç–æ —ç–ª–µ–º–µ–Ω—Ç –µ—â—ë –Ω–µ –≤—Å—Ç—Ä–µ—á–∞–ª—Å—è.
* –°–æ—Ö—Ä–∞–Ω—è–µ—Ç—Å—è –ø–æ—Ä—è–¥–æ–∫.

```python
lst = [1, 2, 2, 3, 4, 3]
unique_lst = []
for item in lst:
    if item not in unique_lst:
        unique_lst.append(item)
print(unique_lst)  # [1, 2, 3, 4]
```

### **4. –ß–µ—Ä–µ–∑ `collections.OrderedDict`** (–¥–ª—è —Å—Ç–∞—Ä—ã—Ö –≤–µ—Ä—Å–∏–π Python)
