### **3. –û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã –∫–æ—Ä—Ç–µ–∂–µ–π**

* –ö–æ—Ä—Ç–µ–∂–∏ –∏–º–µ—é—Ç **–æ—á–µ–Ω—å –º–∞–ª–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤**, —Ç–∞–∫ –∫–∞–∫ –æ–Ω–∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã:

| –ú–µ—Ç–æ–¥                            | –û–ø–∏—Å–∞–Ω–∏–µ                                                             |
| -------------------------------- | -------------------------------------------------------------------- |
| `tuple.count(value)`             | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π –∑–Ω–∞—á–µ–Ω–∏—è                             |
| `tuple.index(value, start, end)` | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ –≤—Ö–æ–∂–¥–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏—è, –º–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –¥–∏–∞–ø–∞–∑–æ–Ω |

```python

```

* –î–ª—è –æ—Å—Ç–∞–ª—å–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç **—Ñ—É–Ω–∫—Ü–∏–∏ –∏ –æ–ø–µ—Ä–∞—Ü–∏–∏**, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–º–µ–Ω–∏–º—ã –∫ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è–º: `len()`, `in`, `+`, `*`, `min()`, `max()`, `sum()`.

