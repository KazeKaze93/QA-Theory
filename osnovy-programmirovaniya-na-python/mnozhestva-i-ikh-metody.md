### **3. –ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è**

```python

# –î–æ–±–∞–≤–ª–µ–Ω–∏–µ
a.add(10)

# –£–¥–∞–ª–µ–Ω–∏–µ
a.discard(2)
# a.remove(100)  # KeyError, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–∞ –Ω–µ—Ç

# –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏
print(a | b)  # {1, 3, 4, 5, 10}  - –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ
print(a & b)  # {3}              - –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ
print(a - b)  # {1, 10}          - —Ä–∞–∑–Ω–æ—Å—Ç—å
print(a ^ b)  # {1, 4, 5, 10}    - —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å

# –ü—Ä–æ–≤–µ—Ä–∫–∞ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞
```
