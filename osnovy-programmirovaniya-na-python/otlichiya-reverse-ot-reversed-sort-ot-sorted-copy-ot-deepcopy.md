### **1. `reverse()` vs `reversed()`**

| –ú–µ—Ç–æ–¥ / –§—É–Ω–∫—Ü–∏—è  | –ß—Ç–æ –¥–µ–ª–∞–µ—Ç                                          | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç | –ò–∑–º–µ–Ω—è–µ—Ç –∏—Å—Ö–æ–¥–Ω—ã–π –æ–±—ä–µ–∫—Ç?         |
| ---------------- | --------------------------------------------------- | ---------- | --------------------------------- |
| `list.reverse()` | –†–∞–∑–≤–æ—Ä–∞—á–∏–≤–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –Ω–∞ –º–µ—Å—Ç–µ                       | `None`     | –î–∞                                |
| `reversed(seq)`  | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏—Ç–µ—Ä–∞—Ç–æ—Ä —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –≤ –æ–±—Ä–∞—Ç–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ | –ò—Ç–µ—Ä–∞—Ç–æ—Ä   | –ù–µ—Ç (–∏—Å—Ö–æ–¥–Ω—ã–π –æ–±—ä–µ–∫—Ç –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è) |

**–ü—Ä–∏–º–µ—Ä:**

```python
lst = [1, 2, 3]

lst.reverse()
print(lst)  # [3, 2, 1]

lst2 = [1, 2, 3]
rev = reversed(lst2)
print(list(rev))  # [3, 2, 1]
print(lst2)       # [1, 2, 3]  - –∏—Å—Ö–æ–¥–Ω—ã–π —Å–ø–∏—Å–æ–∫ –Ω–µ –∏–∑–º–µ–Ω–∏–ª—Å—è
```

### **2. `sort()` vs `sorted()`**

| –ú–µ—Ç–æ–¥ / –§—É–Ω–∫—Ü–∏—è    | –ß—Ç–æ –¥–µ–ª–∞–µ—Ç                              | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç   | –ò–∑–º–µ–Ω—è–µ—Ç –∏—Å—Ö–æ–¥–Ω—ã–π –æ–±—ä–µ–∫—Ç?         |
| ------------------ | --------------------------------------- | ------------ | --------------------------------- |
| `list.sort()`      | –°–æ—Ä—Ç–∏—Ä—É–µ—Ç —Å–ø–∏—Å–æ–∫ –Ω–∞ –º–µ—Å—Ç–µ               | `None`       | –î–∞                                |
| `sorted(iterable)` | –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤—ã–π –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ | –ù–æ–≤—ã–π —Å–ø–∏—Å–æ–∫ | –ù–µ—Ç (–∏—Å—Ö–æ–¥–Ω—ã–π –æ–±—ä–µ–∫—Ç –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è) |

**–ü—Ä–∏–º–µ—Ä:**


### **3. `copy()` vs `deepcopy()`**

| –§—É–Ω–∫—Ü–∏—è              | –ß—Ç–æ –¥–µ–ª–∞–µ—Ç                              | –û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏                                                                        |
| -------------------- | --------------------------------------- | ---------------------------------------------------------------------------------- |
| `copy.copy(obj)`     | –°–æ–∑–¥–∞—ë—Ç **–ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω—É—é –∫–æ–ø–∏—é** –æ–±—ä–µ–∫—Ç–∞ | –ï—Å–ª–∏ –æ–±—ä–µ–∫—Ç —Å–æ–¥–µ—Ä–∂–∏—Ç –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã, –æ–Ω–∏ **–Ω–µ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è**, –∞ –æ—Å—Ç–∞—é—Ç—Å—è —Å—Å—ã–ª–∫–∞–º–∏ |
| `copy.deepcopy(obj)` | –°–æ–∑–¥–∞—ë—Ç **–≥–ª—É–±–æ–∫—É—é –∫–æ–ø–∏—é** –æ–±—ä–µ–∫—Ç–∞      | –í–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã —Ç–æ–∂–µ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è, –ø–æ–ª–Ω–æ—Å—Ç—å—é –Ω–µ–∑–∞–≤–∏—Å–∏–º–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞                 |

**–ü—Ä–∏–º–µ—Ä:**

```python

lst = [[1, 2], [3, 4]]

shallow = copy.copy(lst)
deep = copy.deepcopy(lst)

shallow[0][0] = 100
print(lst)    # [[100, 2], [3, 4]]  - –≤–ª–æ–∂–µ–Ω–Ω—ã–µ —Å–ø–∏—Å–∫–∏ –æ—Å—Ç–∞–ª–∏—Å—å –æ–±—â–∏–º–∏

deep[1][1] = 200
print(lst)    # [[100, 2], [3, 4]]  - deepcopy –Ω–µ –∏–∑–º–µ–Ω—è–µ—Ç –∏—Å—Ö–æ–¥–Ω—ã–π —Å–ø–∏—Å–æ–∫
```

***

üí° **–í—ã–≤–æ–¥:**

* `reverse()` –∏ `sort()` ‚Äî –º–µ—Ç–æ–¥—ã, **–∏–∑–º–µ–Ω—è—é—Ç –æ–±—ä–µ–∫—Ç –Ω–∞ –º–µ—Å—Ç–µ**.
* `reversed()` –∏ `sorted()` ‚Äî —Ñ—É–Ω–∫—Ü–∏–∏, **–≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç**, –∏—Å—Ö–æ–¥–Ω—ã–π –Ω–µ –º–µ–Ω—è—é—Ç.
* `copy()` ‚Äî –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω–∞—è –∫–æ–ø–∏—è, –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã –æ—Å—Ç–∞—é—Ç—Å—è –æ–±—â–∏–º–∏.
* `deepcopy()` ‚Äî –ø–æ–ª–Ω–∞—è –Ω–µ–∑–∞–≤–∏—Å–∏–º–∞—è –∫–æ–ø–∏—è, –≤–∫–ª—é—á–∞—è –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã.
