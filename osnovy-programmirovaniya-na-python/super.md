### **4. –ü—Ä–∏–º–µ–Ω–µ–Ω–∏–µ –≤ –∞–≤—Ç–æ—Ç–µ—Å—Ç–∞—Ö**

* **Page Object Model**: –Ω–∞—Å–ª–µ–¥–æ–≤–∞–Ω–∏–µ –±–∞–∑–æ–≤–æ–≥–æ –∫–ª–∞—Å—Å–∞ —Å—Ç—Ä–∞–Ω–∏—Ü—ã

```python

    def open(self, url):
        print(f"Opening {url}")

    def open(self, url):
        super().open(url)  # –≤—ã–∑–æ–≤ –±–∞–∑–æ–≤–æ–≥–æ –º–µ—Ç–æ–¥–∞
        print("Filling login form")

```
