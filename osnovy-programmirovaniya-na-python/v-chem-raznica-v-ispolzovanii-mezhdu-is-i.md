### 1. **`==`** ‚Äî —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π (equality)

* –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, **—Ä–∞–≤–Ω—ã –ª–∏ –∑–Ω–∞—á–µ–Ω–∏—è** –æ–±—ä–µ–∫—Ç–æ–≤.
* –í—ã–∑—ã–≤–∞–µ—Ç –º–µ—Ç–æ–¥ `__eq__` –æ–±—ä–µ–∫—Ç–∞.
* –î–∞–∂–µ –µ—Å–ª–∏ –æ–±—ä–µ–∫—Ç—ã —Ä–∞–∑–Ω—ã–µ –≤ –ø–∞–º—è—Ç–∏, –Ω–æ —Å–æ–¥–µ—Ä–∂–∞—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ –¥–∞–Ω–Ω—ã–µ ‚Äî `==` –≤–µ—Ä–Ω—ë—Ç `True`.

```python
a = [1, 2, 3]
b = [1, 2, 3]

print(a == b)  # True  (—Å–ø–∏—Å–∫–∏ —Å –æ–¥–∏–Ω–∞–∫–æ–≤—ã–º —Å–æ–¥–µ—Ä–∂–∏–º—ã–º)
```

***

### 2. **`is`** ‚Äî —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –∏–¥–µ–Ω—Ç–∏—á–Ω–æ—Å—Ç–∏ (identity)

* –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, **—è–≤–ª—è—é—Ç—Å—è –ª–∏ –¥–≤–∞ –∏–º–µ–Ω–∏ —Å—Å—ã–ª–∫–∞–º–∏ –Ω–∞ –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –æ–±—ä–µ–∫—Ç –≤ –ø–∞–º—è—Ç–∏**.
* –°—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç `id(a)` –∏ `id(b)`.

```python
a = [1, 2, 3]
b = a

print(a is b)   # True  (—Å—Å—ã–ª–∞—é—Ç—Å—è –Ω–∞ –æ–¥–∏–Ω –æ–±—ä–µ–∫—Ç)
print(a == b)   # True  (–∏ –∑–Ω–∞—á–µ–Ω–∏—è –æ–¥–∏–Ω–∞–∫–æ–≤—ã)
```

***

### 3. –ü—Ä–∏–º–µ—Ä, –∫–æ–≥–¥–∞ `==` –∏ `is` —Ä–∞–∑–ª–∏—á–∞—é—Ç—Å—è

```python

y = [1, 2, 3]

print(x == y)  # True  (–æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ –¥–∞–Ω–Ω—ã–µ)
print(x is y)  # False (—Ä–∞–∑–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã –≤ –ø–∞–º—è—Ç–∏)
```
