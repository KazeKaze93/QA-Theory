### üîπ –ê–ª–≥–æ—Ä–∏—Ç–º –ø–æ–∏—Å–∫–∞ –±–∏—Ç—ã—Ö —Å—Å—ã–ª–æ–∫

1.  –ü–æ–ª—É—á–∏—Ç—å –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã `<a>` (—Å—Å—ã–ª–∫–∏) –Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü–µ:


2. –î–ª—è –∫–∞–∂–¥–æ–π —Å—Å—ã–ª–∫–∏ –¥–æ—Å—Ç–∞—Ç—å –∞—Ç—Ä–∏–±—É—Ç `href`.
3. –ü—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ —Å—Å—ã–ª–∫–∞ **–Ω–µ –ø—É—Å—Ç–∞** –∏ **–Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å http/https**.
4. –û—Ç–ø—Ä–∞–≤–∏—Ç—å **HTTP-–∑–∞–ø—Ä–æ—Å** –∫ —ç—Ç–æ–º—É URL (–æ–±—ã—á–Ω–æ –º–µ—Ç–æ–¥–æ–º `HEAD` –∏–ª–∏ `GET`).
5. –ü—Ä–æ–≤–µ—Ä–∏—Ç—å HTTP —Å—Ç–∞—Ç—É—Å –∫–æ–¥ –æ—Ç–≤–µ—Ç–∞:
   * `200 OK` ‚Üí —Å—Å—ã–ª–∫–∞ —Ä–∞–±–æ—á–∞—è
   * `404`, `500`, `403` –∏ –¥—Ä. ‚Üí —Å—Å—ã–ª–∫–∞ –ø–æ–≤—Ä–µ–∂–¥—ë–Ω–Ω–∞—è



```python
from selenium import webdriver
import requests

driver = webdriver.Chrome()
driver.get("https://example.com")

# –ù–∞—Ö–æ–¥–∏–º –≤—Å–µ —Å—Å—ã–ª–∫–∏ <a>
links = driver.find_elements("tag name", "a")

for link in links:
    url = link.get_attribute("href")

    if not url:  # –ø—É—Å—Ç–∞—è –∏–ª–∏ None
        print("‚ùå –ü—É—Å—Ç–∞—è —Å—Å—ã–ª–∫–∞")
        continue

    try:
        # HEAD –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º GET (–Ω–µ –∑–∞–≥—Ä—É–∂–∞–µ—Ç —Ç–µ–ª–æ –æ—Ç–≤–µ—Ç–∞)
        response = requests.head(url, allow_redirects=True, timeout=5)
        if response.status_code >= 400:
            print(f"‚ùå –ë–∏—Ç–∞—è —Å—Å—ã–ª–∫–∞: {url} | –ö–æ–¥: {response.status_code}")
        else:
            print(f"‚úÖ –†–∞–±–æ—á–∞—è —Å—Å—ã–ª–∫–∞: {url} | –ö–æ–¥: {response.status_code}")
    except requests.exceptions.RequestException as e:
        print(f"‚ö† –û—à–∏–±–∫–∞ –ø—Ä–∏ –ø—Ä–æ–≤–µ—Ä–∫–µ {url}: {e}")

driver.quit()


```

* –î–æ–±–∞–≤–ª—è–π `timeout` (–∏–Ω–∞—á–µ —Ç–µ—Å—Ç –º–æ–∂–µ—Ç –∑–∞–≤–∏—Å–Ω—É—Ç—å).
* –õ—É—á—à–µ —Å–æ–±–∏—Ä–∞—Ç—å **–∏ `<a>` –∏ `<img>`**, –ø–æ—Ç–æ–º—É —á—Ç–æ –±–∏—Ç–∞—è –∫–∞—Ä—Ç–∏–Ω–∫–∞ ‚Üí —Ç–æ–∂–µ –æ—à–∏–±–∫–∞.
