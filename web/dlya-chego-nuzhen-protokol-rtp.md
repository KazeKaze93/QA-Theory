# Для чего нужен протокол RTP?

**RTP (Real-time Transport Protocol)** — это сетевой протокол, используемый для передачи мультимедийных данных в **режиме реального времени**.

#### Основное назначение:

* Передача **аудио и видео потоков** с минимальной задержкой.
* Используется в **VoIP, видеоконференциях, потоковом видео, онлайн-играх**.

#### Ключевые особенности:

* Работает поверх **UDP** (обычно), что обеспечивает низкую задержку.
* Поддерживает **нумерацию пакетов** — можно восстановить порядок при получении.
* Использует **метки времени** для синхронизации аудио и видео.
* Может работать вместе с **RTCP (RTP Control Protocol)** для контроля качества передачи (задержка, потеря пакетов, джиттер).

#### Пример использования:

* Звонок в Zoom или WhatsApp → RTP передаёт твой голос и видео в реальном времени.
